このガイドでは、開発基盤となる`Node.js`をインストールする方法を説明します。

## Mac OS でのインストール

### 方法 1: Homebrew を使用する

1. Homebrew がインストールされていない場合は、ターミナルを開いて以下のコマンドを実行してください。

   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

   Homebrew のインストールが完了したら、次に進みます。

2. Node.js をインストールするには、次のコマンドをターミナルで実行します。
   ```bash
   brew install node
   ```

### 方法 2: Volta を使用する

Rust製のNode.jsバージョン管理ツールです。  
公式では「手間要らずのJavaScriptツールマネージャー」と説明されています。


#### 通常の方法でインストールする場合

ターミナルを開いて以下のコマンドを実行してください。

   ```bash
   curl https://get.volta.sh | bash
   ```

#### Homebrewでインストールする場合

ターミナルを開いて以下のコマンドを実行してください。

```bash
brew install volta
```

#### PATHを通す

PATHを通す必要があるので、以下のコマンドを実行してください。  
.zshrcに以下を記述します。

```bash
export VOLTA_HOME="$HOME/.volta"
export PATH="$VOLTA_HOME/bin:$PATH"
```

#### インストール確認方法

```bash
volta --version
```

以下のようにVoltaのバージョンが表示されれば完了です。

```bash
1.1.1
```

#### Node.js のインストール

最新バージョンをインストールする場合

```bash
volta install node
```

バージョンを指定してインストールする場合

```bash
volta install node@20.11.0
```

### 方法 3: Node.js の公式サイトからインストーラをダウンロード

1. [Node.js 公式ウェブサイト](https://nodejs.org/)にアクセスします。
2. 推奨されるバージョンのインストーラをダウンロードするために「Download Node.js」をクリックします。
3. ダウンロードしたファイルを開き、画面の指示に従ってインストールを完了します。
Mac OS、Windows、Linuxともに方法は同様です。

バージョンを指定してインストーラをダウンロードする場合は、[こちら](https://nodejs.org/en/download)よりダウンロードが可能です。

### インストール後の確認

インストールが成功したかどうかを確認するために、以下のコマンドをターミナル（Mac）またはコマンドプロンプト（Windows）で実行します。

```bash
node -v
npm -v
```

これらのコマンドは、インストールされた Node.js と npm のバージョンを表示します。出力が表示されれば、インストールは成功しています。
