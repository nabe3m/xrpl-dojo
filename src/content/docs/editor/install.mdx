Visual Studio Code (VS Code)とは、Microsoft社の提供する無償のコードエディタです。

当サイトではVS Codeでの開発を前提に紹介しますので、特にこだわりがなければ、VS Codeでの開発を推奨いたします。

## Mac OSでのインストール

1. [VS Codeの公式ウェブサイト](https://code.visualstudio.com/)にアクセスします。
2. 「Download Mac Universal」をクリックし、インストーラをダウンロードします。
3. ダウンロードした`.zip`ファイルを解凍し、得られた`Visual Studio Code.app`を`Applications`フォルダにドラッグしてインストールします。

## Windowsでのインストール

1. [VS Codeの公式ウェブサイト](https://code.visualstudio.com/)にアクセスします。
2. 「Download Windows」をクリックし、インストーラをダウンロードします。
3. ダウンロードしたインストーラファイル（`.exe`ファイル）を実行し、画面の指示に従ってインストールを進めます。

## インストール後の確認

インストールが完了したら、以下のコマンドをターミナル（MacOS）またはコマンドプロンプト（Windows）で実行して、VS Codeが正しくインストールされたことを確認します。

```bash
code --version
