このガイドでは、`xrpl.js`ライブラリを使用してXRP Ledgerに接続し、基本的なトランザクションを行うプロジェクトの作成方法を説明します。

## 必要なツール

- テキストエディタ（VS Codeが推奨）
- Node.jsのインストール

## ステップ 1: プロジェクトのセットアップ

1. 新しいディレクトリを作成し、そのディレクトリ内で以下のコマンドを実行して、新しいNode.jsプロジェクトを初期化します。

    ```bash
    npm init -y
    ```
2. xrpl.jsライブラリをプロジェクトにインストールします。

    ```bash
    npm install xrpl
    ```
